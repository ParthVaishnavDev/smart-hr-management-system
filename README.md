# Smart HR Management System

SAP BTP ABAP Cloud + SAP Fiori Project

## Project Overview

This project demonstrates end-to-end SAP development using:

- SAP BTP ABAP Environment
- CDS Views
- OData V4 Services
- SAP Fiori Elements
- Eclipse ADT

## Features

### Employee Management

- Employee Master Data
- Department Mapping
- Designation Mapping

### CDS Data Model

- Interface Views
- Associations
- Consumption Views
- UI Annotations

### Service Exposure

- Service Definition
- Service Binding
- OData V4 API

### Fiori Elements

- List Report
- Object Page
- Search
- Filters

## Project Architecture

Database Tables
↓
CDS Interface Views
↓
Associations
↓
Consumption View
↓
OData V4 Service
↓
Fiori Elements Application
